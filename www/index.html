www.metaweather.com

MetaWeather is an automated weather data aggregator that takes the weather predictions from various forecasters and calculates the most likely outcome.

The MetaWeather is based on the GET method with a great API and doesn't need any auth token.

Using CORS Proxy, you can use MetaWeather with GET, POST or PUT method and the same API.
